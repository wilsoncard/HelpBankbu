Mercancías

Esta opción está diseñada para aquellas entidades que poseen y administran sus propios comercios y tarjetas de crédito, como pueden ser eventualmente las grandes cadenas de almacenes. Mediante esta función se determinan de manera precisa las diferentes mercancías que van a ser incluidas en los programas de promociones diseñados y mercancías que están asociadas con los atributos definidos. Cuenta con un máximo de tres niveles para las jerarquías o categorías de mercancías establecido.

El formulario contiene las opciones Detalle y Mercancías.

Mercancías: Si el usuario invoca esta opción, se despliega un formulario con las opciones Actualizar, Eliminar, y Adicionar. Asi mismo cuenta con un botón para regresar a la opción anterior.

Adicionar: Si el usuario invoca esta opción, se despliega un formulario con los siguientes campos:

 

Código artículo

Campo alfanumérico de máximo 30 posiciones, obligatorio, en el cual se registra la identificación de cada uno de los artículos definidos.

Descripció artículo

Campo alfanumérico de máximo 60 posiciones, obligatorio, en el que se registra el nombre o descriptivo del artículo asociado al código anterior.

 

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es: Descripción artículo.
