Ordenamiento de casos de uso

Desde este formulario, se habilita el orden el cual quiere el usuario manejar dentro del árbol de menú para los Roles de negocio Núcleo y Emisor, junto con sus respectivos menú y sub-menús, de esta manera permite tener un mejor panorama al usuario de utilidad para la navegación del aplicativo.

.

 

 

Descripción de Campos

Rol de negocio

Contiene los diferentes roles para los cuales está diseñado BANKBU®  y que puede ser núcleo o emisor. En caso de existir personalizaciones para el cliente, se podrá mostrar un rol adicional denominado 'Cliente' en el cual se desplegarán las respectivas opciones.

Opciones de menú

Dependiendo del Rol seleccionado se muestran opciones diferentes en este menú, las cuales permiten el acceso a las funciones del aplicativo de acuerdo con la atribuciones y restricciones de cada perfil

Opciones de sub-menú

Una vez que el usuario ha seleccionado un rol y se despliegan el menú correspondiente, al ubicarse sobre cualquiera de ellos y hacer clic se desplegará el menú que muestra todas las opciones contenidas y a las cuales puede ingresar el usuario.
