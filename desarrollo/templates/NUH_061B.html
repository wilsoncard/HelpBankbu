Sub transacciones/Identificador Red

Mediante esta opción se facilita la consulta y/o mantenimiento, reservado a funcionarios de Open Card S.A., de la tabla en la que se parametriza un atributo más de las transacciones, que permite diferenciarlas o puede ser utilizado para identificar la red a través de la cual se realizó la transacción o el medio por el cual se efectuó, tal como audio, etc. Esta información puede ser utilizada entre otras opciones para dirigir el movimiento a una transacción determinada en el traductor de transacciones.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario

Descripción de campos

Código

Campo numérico de 5 dígitos, obligatorio, asociado a cada uno de los diversos identificadores de red.

Descripción

En este campo alfanumérico, obligatorio, se registra el nombre asociado a cada código.

 

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es Descripción.
