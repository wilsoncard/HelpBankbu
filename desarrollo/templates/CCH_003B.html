Tipos de documentos

Una vez se inicie el proceso de cobranza, la base del mismo la constituyen los documentos recibidos tanto en calidad de garantía como de medio de pago, por lo que esta forma o pantalla permite parametrizar dichos documentos.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar.

 

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

 

Código

Campo alfanumérico de 3 posiciones, obligatorio, asignado a cada una de las clases o tipos de documentos bien sean de garantía o de medio de pago que circulan durante el proceso de cobranza.

Descripción

Este campo alfanumérico de 50 posiciones, obligatorio, contiene el nombre o descriptivo asociado a cada uno de los del tipo de documento.

Cantidad de copias

En este capo se ingresa el número de copias que deben ser entregadas del documento solicitado.
