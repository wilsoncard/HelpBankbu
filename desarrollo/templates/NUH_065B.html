Procesos de validación para establecimientos

Mediante esta función se permite el mantenimiento y/o consulta, a los funcionarios de Open Card S.A., de la tabla en la cual se relacionan todas las posibles homologaciones a realizar en la Validación cargue establecimientos.

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

 

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

 

Código

Campo numérico de 3 dígitos, obligatorio, que identifica cada una de las posibles homologaciones que pueden darse en la validación cargue establecimientos.

Descripción

En este campo alfanumérico de 60 posiciones, obligatorio, se registra el nombre asociado a cada código.

 

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es: Descripción.
