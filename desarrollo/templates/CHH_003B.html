Canales

Mediante esta opción se habilita la consulta y mantenimiento de la tabla en la cual se almacenan los canales o medios que se encuentran habilitados y que son utilizados por los tarjeta-habientes para realizar operaciones de pagos y consultas. Cada canal contiene un código, una descripción y un horario hábil de funcionamiento.
El formulario contiene las opciones adicionar, actualizar, eliminar y el link “Transacciones asociadas al canal”. Adicionalmente contiene un filtro que le permite al usuario consultar información de canales, de acuerdo a un código y/o descripción ingresados.

Código:

Campo numérico de 3 posiciones, que corresponde al identificador del canal.
Descripción: Campo alfanumérico que corresponde al nombre del canal.

Adicionar: si el usuario invoca la opción adicionar se despliega el siguiente formulario:

Código:

Campo numérico de 3 posiciones, obligatorio, que corresponde al identificador del canal.
Descripción: En este campo alfanumérico de 80 posiciones, obligatorio, se registra el nombre del canal. Ejemplo: Internet, Datafono.
Horario hábil de funcionamiento: Campo alfanumérico de 22 posiciones, no obligatorio, que describe el horario de funcionamiento del canal.

Actualizar: Si el usuario invoca la opción Actualizar se despliega el siguiente formulario:

El campo código no se puede modificar.

Transacciones asociadas al canal: Si el usuario invoca la opción Transacciones asociadas al canal se despliega el siguiente formulario:

Mediante esta opción se habilita la consulta de transacciones financieras y no financieras asociadas al canal o medio permitido seleccionado. Esta pantalla permite ver la información correspondiente al Canal y a las transacciones. Cada transacción contiene un Nombre de Transacción, un Tipo, un Indicador de Soporte por el autorizador y un indicador de reversión.

El formulario contiene las opciones adicionar y eliminar.

 

Adicionar: si el usuario invoca la opción adicionar se despliega el siguiente formulario:

Transacción: Campo obligatorio con lista de valores de la cuál se selecciona la transacción que va a ser asociada al canal.
