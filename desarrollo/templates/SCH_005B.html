Novedad datos de identificación

Este formulario invocado desde la consulta de clientes, permite corregir errores de datos en la captura o modificaciones del tipo y/o número del documento de identificación, nombres y apellidos o razón social tanto para personas naturales como jurídicas. Al ingresar al formulario, por defecto despliega los datos actuales registrados para el cliente con el fin que el usuario pueda modificar únicamente aquellos datos que se requiera actualizar.

Cuando el cliente es una persona natural se despliegan los siguientes campos:

 

Si el cliente es una persona jurídica los campos que muestra el formulario con los siguientes.

 

Descripción de campos

Nombres

Si se trata de una persona Natural se dispone de dos campos, el primero de los cuales es obligatorio y en los que se ingresa el o los nombres correctos del cliente. Cuando se trata de una persona Jurídica muestra un único campo obligatorio en el cual se puede registrar el nombre completo de la misma.

Apellidos

Este campo que se muestra únicamente para los clientes naturales, dispone de dos campos, el primero de los cuales es obligatorio, y en los que se puede ingresar el o los apellidos del cliente.

Tipo identificación

Campo obligatorio tipo combo del cual se puede seleccionar el tipo de documento de identificación correspondiente al cliente.

Tipo persona

En este campo de salida, se muestra de manera automática el tipo de persona asociada con el documento seleccionado en el campo anterior.

Número identificación

Campo alfanumérico obligatorio de hasta 16 caracteres en el cual se registra el número del documento del cliente.

Sexo

En este campo tipo combo que solo se muestra para los clientes naturales y que es obligatorio, se puede seleccionar entre Masculino o Femenino el sexo al cual pertenece el cliente.

Fecha radicación

Campo obligatorio en el que se registra en formato YYYY-MM-DD la fecha en la cual el cliente reportó el cambio de datos y que puede ser una fecha anterior a la actual del sistema, facilitando el ingreso de novedades de fechas anteriores.
