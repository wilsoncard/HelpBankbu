Bloqueos

Mediante esta función se habilita la consulta y mantenimiento (reservado a funcionarios de Open Card S.A.), de la tabla en la cual se almacenan los códigos, descripciones y tipos de las diferentes condiciones o estados que puede tener un crédito. La forma y condiciones bajo las cuales se aplica cada uno de estos géneros de bloqueo las determina la entidad financiera en la opción Estados y bloqueos.

 

El formulario contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario

Descripción de campos

Código

Campo alfanumérico de 1 posición, obligatorio, con el que se identifica cada bloqueo a definir. Este campo no es modificable, aún en Modo Total, por lo cual para corregir un código errado es necesario eliminar el registro y crearlo nuevamente.

Descripción

En este campo alfanumérico de 30 posiciones, obligatorio, se registra el nombre o descripción de cada bloqueo codificado.

Tipo bloqueo

Campo que posee lista de valores adjunta de la que debe seleccionarse entre Reestructuración, Plástico, Sobrecupo, Persona o Saldo la categoría asociable a cada bloqueo definido.

 

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificables son Descripción y Tipo bloqueo
