Consulta Cargue de Autorizaciones Débito

Mediante esta función la entidad realiza la actualización de las autorizaciones de utilización otorgadas a los clientes débito.

Filtro: El formulario cuenta con un filtro inicial de búsqueda, necesario para la consulta de la información:

 

Número autorización

En este campo se captura el número de autorización sobre el cual se desea realizar la consulta.

Número tarjeta

En este campo se captura el número de tarjeta del cliente a quien se desea consultar.

Código respuesta

En este campo se captura el código de respuesta correspondiente a la autorización sobre la cual se desea realizar la consulta.

Fecha transacción

Este campo acepta caracteres en formato AAAA-MM-DD

Hora transacción

Este campo acepta caracteres en formato HH:MM:SS

Clase transacción

En este campo se captura la clase de transacción que se desea consultar.

Fecha cargue

Este campo acepta caracteres en formato AAAA-MM-DD
Marca Permite filtrar por marca las autorizaciones de utilizaciones de clientes débito

 

A través del botón Buscar, se muestra un formulario que contiene la siguiente información, junto con la opción Detalle.

 

Detalle: A través de este botón el sistema muestra un formulario con la siguiente información. Adicionalmente, cuenta con la opción Código rechazo.

 

Código rechazo: A través de este botón el sistema muestra un formulario con el código y la descripción de los rechazos presentados en el cargue de la autorización consultada. Si la autorización no presentó rechazos, el formulario no mostrará información.
