Definiciòn de productos - Productos amparados

 

Esta opción que se invoca desde el formulario Definición de productos, le permite a la entidad adicionar o modificar la información correspondiente a aquellos otros productos de la entidad que pueden ser entregados en calidad de amparados (extendidas, adicionales o amparadas) y que pueden depender directamente de una tarjeta perteneciente al producto que se está parametrizando. Si para un Producto no se registra información en este formulario, implícitamente se está indicando que para este no sería permitido solicitar ningún tipo de tarjeta dependiente.

 

El formulario contiene las opciones Eliminar y Adicionar.

Adicionar: si el usuario invoca la opción Adicionar se despliega un nuevo formulario con los siguientes campos.

 

Descripción de campos

Producto amparado

Campo obligatorio con lista de valores de la cual se puede seleccionar el producto que puede ser amparado por un titular de una tarjeta perteneciente al producto que se está parametrizando.
