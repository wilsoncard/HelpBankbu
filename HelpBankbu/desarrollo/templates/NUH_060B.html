Perfiles promotor

Mediante esta función se posibilita el codificar el nivel de responsabilidad de cada uno de los asesores (promotores) dentro de la entidad, información que es utilizada entre otros, en la parametrización en la opción Funcionarios. El nivel asignado a cada asesor determina las áreas de intervención que éste puede realizar.

El formulario solo contiene las opciones Actualizar, Eliminar y Adicionar.

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario.

Descripción de campos

Código

Identificación asignada a cada nivel de promotor. Es un campo alfanumérico de 16 posiciones y su diligenciamiento es obligatorio.

Descripción

De acuerdo con la codificación asignada a cada nivel de promotor, en este campo se detalla el nombre de dicho nivel de acuerdo con su código. Campo obligatorio.

 

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual el único campo modificable es Descripción.
