Mensajes del extracto

Mediante esta función se habilita la consulta y mantenimiento de la tabla que contiene la información que la entidad desea trasmitirle a sus clientes. Existe una jerarquía de los tipos de mensaje, el sistema despliega primero los asociados a los bloqueos, luego los generados por la segmentación de clientes y finalmente los relacionados con los productos.  La forma ofrece tres líneas por cada clase de mensaje, espacio del que se dispone pero es necesario optimizar y adecuar el tipo y tamaño de letra con la cantidad de caracteres a utilizar.

El formulario contiene las opciones Actualizar, Eliminar, Adicionar y Detalle.

 

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes campos:

 

Código

Campo numérico de 4 dígitos, obligatorio, que contiene el número que identifica a cada clase de mensaje que aparece dentro del extracto del cliente.

Descripción

En este campo alfanumérico de 30 dígitos, se registra el nombre asociado al código de cada mensaje.

Moneda

Campo obligatorio que posee lista de valores de la que debe seleccionarse la Moneda en que se expresan los valores del extracto al que se le asocia el correspondiente mensaje.

Tipo

En este campo obligatorio se debe seleccionar de la lista de valores adjunta de la cual se debe seleccionar entre Financiero, Comercial, General o Bloqueo el tipo de mensaje.

Línea 1

Campo alfanumérico de 240 dígitos, no obligatorio, dentro del que se registra el contenido de cada una de las líneas que para un mismo tipo de mensaje contendrá el extracto.

Línea 2

Campo alfanumérico de 240 dígitos, no obligatorio, dentro del que se registra el contenido de cada una de las líneas que para un mismo tipo de mensaje contendrá el extracto.

Línea 3

Campo alfanumérico de 240 dígitos, no obligatorio, dentro del que se registra el contenido de cada una de las líneas que para un mismo tipo de mensaje contendrá el extracto.

 

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificables son: Descripción, Tipo, Línea 1, Línea 2, y Línea 3.
