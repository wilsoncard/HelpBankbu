Condiciones de afiliación

 

Por medio de esta opcion la entidad define las condiciones establecidas para los diferentes tiós de establecimientos.

El formulario contiene las opciones Actualizar, Eliminar, y Adicionar. Adicionalmente, cuenta con un filtro de búsqueda.

Filtro: Se pueden realizar consultas a través de las siguientes opciones:

Código

Campo alfanumérico de máximo 6 caracteres, en el que se registra el código asignado a cada uno de las condiciones.

Descripción

Este campo alfanumérico de 30 posiciones, permite ingresar el nombre de la condición.

Periodo a evaluar

Lista de valores que permite seleccionar una de las siguientes opciones: Año, Semestre, Trimestre, Bimestre, y Mes.

 

Adicionar: Si el usuario invoca la opción Adicionar se despliega un nuevo formulario.

 

Descripción de campos

Código de afiliación

Campo alfanumérico de máximo 6 caracteres, en el que se registra el código asignado a cada uno de las condiciones.

Descripción

Este campo alfanumérico de 30 posiciones, permite ingresar el nombre de la condición.

Monto mínimo

Para este campo el sistema admite cualquier dato numérico de máximo 14 dígitos, 12 enteros y dos decimales.

Monto máximo

Para este campo el sistema admite cualquier dato numérico de máximo 14 dígitos, 12 enteros y dos decimales. El monto máximo debe ser mayor al monto mínimo..

Monto de penalización

Para este campo el sistema admite cualquier dato numérico de máximo 14 dígitos, 12 enteros y dos decimales.

Máximo número de insatisfacción

Lista de valores que permite ingresar cualquier valor numérico de máximo 3 dígitos.

Periodo a evaluar

Lista de valores que permite seleccionar una de las siguientes opciones: Año, Semestre, Trimestre, Bimestre, y Mes.

 

Actualizar: Si el usuario invoca la opción Actualizar se despliega un nuevo formulario en el cual los únicos campos modificables son: Descripción, Monto mínimo, Monto máximo, Monto de penalización, Máximo número de insatisfacción, y Periodo a evaluar.

 

Detalle: Si el usuario invoca la opción Detalle se despliega el siguiente formulario.
