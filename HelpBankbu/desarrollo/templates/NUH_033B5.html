Franquicias

Parámetros acciones por bloqueos: Grupo de datos que permite relacionar los bloqueos que pueden presentar las tarjetas o los clientes y las acciones o causales por las que se generan las novedades en el exception file.

El formulario contiene las opciones Eliminar y Adicionar. Adicionalmente, cuenta con un botón en la parte superior para regresar al formulario principal de Franquicias.

 

Adicionar: Si el usuario invoca la opción Adicionar se despliega un formulario con los siguientes bloques:

 

Bloqueo

Campo que posee lista de valores poblada a partir de la opción de Estados y bloqueos, de la cual toma únicamente los bloqueos en los cuales dentro del grupo de transacciones habilitadas el campo Exception file esté marcado como activo o que aplica.

Parámetros acciones para el exception file

Lista de valores que muestra las acciones parametrizadas en la opción cn el mismo nombre perteneciente a este mismo caso.
